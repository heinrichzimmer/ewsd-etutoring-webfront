# EWSD-Etutoring-Webfront
## Branches
- `main` - The stable version of the project. Only merged from `develop`.
- `develop`- The main integration branch for features and bug fixes.
- `feat-{name}` - New features or enhancements (`feat-login-api`).
- `fix-{issue}` - Fixing bugs found in the development branch.

## Libraries
-  `shadcn` - UI Library
-  `date-fns` - Date Formatter
-  `react-query` - Client-side Data Fetching Library
